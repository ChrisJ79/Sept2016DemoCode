New Repository:
1.) Create new repo in github, and copy the url it gives you
2.) cd to root directory of repo
3.) git init
4.) git remote add origin <url>
5.) git add -A
6.) git commit -m "your message"
7.) git push origin master

Typical use to push up work:
1.) git add -A
2.) git commit -m "your message"
3.) git push origin master

To get new files in a repo (for example to get new notes from demo code repo):
1.) cd to root of repo
2.) git pull origin master

Note: If you clone the exercise-starters repo, make sure to copy the files you need into your own repo since you will not have access to push to the exercise-starters repo.
